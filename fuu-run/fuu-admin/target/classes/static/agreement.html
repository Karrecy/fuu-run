一、定义

  • 跑腿服务：指用户通过本平台发布任务（如代取快递、购买物品、送餐、取快递等），并由平台指定或自愿接单的跑腿人员为其提供服务的行为。
  • 用户：指注册并使用本平台服务的个人或机构。
  • 跑腿人员：指在本平台上注册成为跑腿人员、接受用户任务并为用户提供跑腿服务的个人。
  • 校园平台：指以高校或校园为主要服务区域的跑腿平台系统。

二、服务内容

1. 本平台提供校园内跑腿、代取、代送等综合服务，供用户发布任务并由跑腿人员接单完成。

2. 用户可以通过本平台提交任务信息，包括但不限于任务描述、取送地址、时限要求、报酬等。

3. 跑腿人员根据任务内容选择接单，并与用户就任务细节进行确认，本平台作为信息发布和任务撮合平台，不直接介入任务履行过程。

三、用户注册与账户安全

四、任务发布与接单

2. 跑腿人员在接单前应核实任务信息，并与用户就任务要求、地点、时间等进行充分确认；如因沟通不畅引发纠纷，由双方协商解决，本平台有权根据任务情况介入协调。

3. 用户与跑腿人员应互相尊重，保持诚信，不得利用任务信息进行不正当竞争或商业炒作。

五、费用结算与报酬

1. 用户发布任务时应明确约定任务报酬及支付方式，本平台将根据任务约定扣除相应手续费后进行结算。

2. 用户和跑腿人员在任务完成后应及时确认任务状态，确认无误后进行费用结算；若发生争议，本平台将依据平台规则进行调解处理。
